# Café Central

O Café Central é um projeto de site para uma cafeteria, desenvolvido com o objetivo de apresentar a empresa, seus produtos e facilitar a interação com os clientes através de funcionalidades como cadastro, login e contato.

## Sobre o Projeto

O site da Café Central oferece uma experiência simples e agradável para os usuários conhecerem a cafeteria, visualizarem o cardápio e entrarem em contato com a equipe.

## Funcionalidades

### Página Inicial (Index)
- Apresentação da cafeteria.
- Destaques dos produtos e serviços.

### Cadastro de Usuários
- Registro de novos clientes.

### Login
- Autenticação de usuários cadastrados.

### Cardápio
- Exibição dos produtos disponíveis.
- Informações sobre cafés, bebidas e acompanhamentos.

### Contato
- Formulário para envio de mensagens.
- Canal de comunicação com a cafeteria.

### Sobre Nós
- História da Café Central.

## Tecnologias Utilizadas

## Como Executar o Projeto

1. Clone este repositório:

```bash
git clone https://github.com/seu-usuario/cafe-central.git
```

2. Acesse a pasta do projeto:

```bash
cd cafe-central
```

3. Abra o arquivo `index.html` em seu navegador.
